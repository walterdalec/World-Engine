World Engine — Portrait Starter Pack (Placeholders)
===================================================
This pack contains transparent PNG placeholders (1024×1024) aligned for the
CharacterCreatorApp. Replace any file with final art using the same filename.

Folders
-------
- base/male/neutral.png
- base/female/neutral.png
- race/*.png (overlays for species features)
- class/*.png (collar/pauldron/robe overlays)
- deco/*.png (optional extras)
- catalog.json (optional, matches code schema)

Notes
-----
- These are *placeholders* for alignment & testing, not final painterly art.
- Old-school CRPG vibe: keep head-and-shoulders framing consistent.
- Final art can be sex-specific by moving files into /race/{sex}/ and /class/{sex}/.
